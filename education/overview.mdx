# Educational Background

Álvaro Navarro Pegalajar is currently pursuing his studies in **Multiplatform Application Development** (Desarrollo de Aplicaciones Multiplataforma - DAM) at IES Oretania in Linares, Jaén, Spain.

## IES Oretania

<Info>
IES Oretania is a vocational training institution located in Linares, Jaén, offering specialized technical education in various fields including information technology and software development.
</Info>

The institute provides comprehensive training programs designed to prepare students for real-world software development careers, combining theoretical knowledge with practical hands-on experience.

## Multiplatform Application Development Program

The **Desarrollo de Aplicaciones Multiplataforma (DAM)** program is a higher-level vocational training course (Ciclo Formativo de Grado Superior) that prepares students to design, develop, and deploy applications across multiple platforms.

### Core Competencies

The DAM program develops professional competencies in several key areas:

<Accordion title="Database Design and Management">
Students learn to design, implement, and manage relational and non-relational databases, including SQL optimization, data modeling, and database administration.
</Accordion>

<Accordion title="Business Application Programming">
Development of enterprise-level applications using object-oriented programming paradigms, design patterns, and best practices for maintainable code.
</Accordion>

<Accordion title="User Interface Development">
Creating intuitive and responsive user interfaces for desktop, web, and mobile applications, with emphasis on user experience (UX) design principles.
</Accordion>

<Accordion title="Multimedia and Mobile Development">
Building native and cross-platform mobile applications for Android and iOS, including multimedia integration and device-specific features.
</Accordion>

<Accordion title="Data Access Programming">
Implementing data persistence layers, ORM frameworks, API integration, and efficient data management strategies.
</Accordion>

<Accordion title="Business Process Management">
Understanding software development within business contexts, including project management, agile methodologies, and stakeholder communication.
</Accordion>

<Accordion title="Systems Management">
Configuration and administration of development environments, version control systems, CI/CD pipelines, and deployment infrastructure.
</Accordion>

### Learning Approach

<Note>
The program emphasizes practical, project-based learning where students work on real-world scenarios and build portfolio-worthy applications throughout their studies.
</Note>

The curriculum is designed to:

- **Integrate Theory and Practice**: Every theoretical concept is reinforced through hands-on coding exercises and projects
- **Foster Problem-Solving Skills**: Students tackle increasingly complex challenges that mirror industry demands
- **Encourage Collaboration**: Team projects simulate real development team dynamics
- **Stay Current**: Regular curriculum updates ensure students learn modern technologies and methodologies

### Professional Internship

The final phase of the program includes a mandatory professional internship (Formación en Centros de Trabajo - FCT) where students:

- Work in real companies or organizations
- Apply their acquired skills to actual projects
- Experience professional development workflows
- Build industry connections and networking opportunities

<Tip>
The internship period is crucial for transitioning from academic learning to professional practice, often leading to job offers upon completion.
</Tip>

## Academic Philosophy

Álvaro's educational journey at IES Oretania emphasizes:

1. **Continuous Learning**: Staying updated with emerging technologies and industry trends
2. **Quality Code**: Writing clean, maintainable, and well-documented code
3. **User-Centric Design**: Prioritizing end-user needs and experience
4. **Collaborative Development**: Working effectively in team environments
5. **Professional Ethics**: Understanding the responsibilities of software developers in society

## Career Preparation

Upon completion of the DAM program, graduates are qualified to work as:

- Multiplatform Application Developers
- Mobile Application Developers (Android/iOS)
- Full-stack Web Developers
- Database Administrators
- Software Quality Assurance Specialists
- Technical Project Coordinators

<Info>
The DAM qualification is recognized throughout the European Union, providing opportunities for international career development.
</Info>
